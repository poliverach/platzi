Stylus
------
  - Download nodejs (https://nodejs.org/)
  - Install
  - Run npm install stylus -g

  Install "nib" ()
  -------------
    - npm install nib

    Docs
    ----
      - http://nibstyl.us/docs/

  Convert to CSS
  -------------
    - normal:
        stylus -c -o css/ stylus/estilos.styl
    - watching
        stylus -w -c -o css/ stylus/estilos.styl
    - use "nib"
        stylus -u nib -c -o css/ stylus/estilos.styl

  Install css2stylus (https://www.npmjs.com/package/css2stylus)
  ------------------
    - npm install -g css2stylus
